# 바이브코드 AI 웹 랜딩페이지

AI와 함께하는 실전 웹 개발 클래스의 공식 랜딩페이지입니다.

## 📁 프로젝트 구조

```
vibe-coding-landing/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 스타일시트
├── js/
│   └── script.js      # JavaScript 기능
└── README.md          # 프로젝트 설명서
```

## 🎨 디자인 특징

- **전문적이고 트렌디한 디자인**: 현대적인 그라데이션과 섹션 구분
- **반응형 웹 디자인**: 모바일, 태블릿, 데스크톱 모든 화면 크기 지원
- **부드러운 애니메이션**: 스크롤 애니메이션과 호버 효과
- **가독성 중심**: 적절한 글자 크기와 색상 대비

## 🛠️ 기술 스택

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)

## ✨ 주요 기능

1. **고정 헤더**: 스크롤 시 배경 변화
2. **스무스 스크롤**: 앵커 링크 클릭 시 부드러운 이동
3. **모바일 메뉴**: 반응형 햄버거 메뉴
4. **스크롤 애니메이션**: 요소가 화면에 나타날 때 페이드인 효과
5. **진행 표시바**: 페이지 상단의 스크롤 진행도 표시

## 🚀 사용 방법

1. 모든 파일을 웹 서버에 업로드하거나
2. `index.html` 파일을 브라우저에서 직접 열기

## 📱 반응형 브레이크포인트

- 데스크톱: 1024px 이상
- 태블릿: 768px ~ 1023px
- 모바일: 767px 이하

## 🎨 색상 팔레트

- Primary: #6366f1 (인디고)
- Secondary: #ec4899 (핑크)
- Success: #10b981 (그린)
- Warning: #f59e0b (오렌지)

## 📝 섹션 구성

1. **히어로 섹션**: 메인 메시지와 CTA 버튼
2. **바이브코딩이란**: 개념 설명 및 차별점
3. **교육 목표**: 핵심 학습 목표
4. **기술 스택**: 사용 기술 소개
5. **커리큘럼**: 기초/심화 과정 타임라인
6. **기대효과**: 장점과 한계점 명시
7. **CTA**: 수강 신청 유도
8. **푸터**: 브랜드 정보

## 🔧 커스터마이징

### 색상 변경
`css/style.css` 파일의 `:root` 변수를 수정하세요:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    /* 원하는 색상으로 변경 */
}
```

### 콘텐츠 수정
`index.html` 파일에서 텍스트 내용을 직접 수정하세요.

### 애니메이션 조정
`js/script.js` 파일에서 타이밍과 효과를 조정할 수 있습니다.
